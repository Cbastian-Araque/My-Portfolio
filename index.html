Sobre mí

Hola! Soy sebastián Araque, tengo experiencia en desarrollo web utilizando HTML5 y CSS3, también tengo conocimientos en JavaScript y algunas librerías para diseño. He utilizado NodeJs para crear API REST. Todo el conocimiento lo he ido adquiriendo de manera autodidacta, con ayuda de Youtube, Google, Udemy, Foros, y mucha motivación. Tengo experiencia en maquetación web, diseño web, UI, uso de Git y GitHub, también algunas librerías utiles para JavaScript como, ScrollReveal y Typed.js.
